# HOW TO RUN THE CODE - Quick Reference Guide

## Step 1: Compile the Project
```bash
cd d:\23i0727-23i0661-F
make clean
make
```

After successful compilation, you'll have `slr_parser.exe` in the root directory.

## Step 2: Run the Parser

### Basic Syntax
```bash
.\slr_parser.exe <grammar_file> <input_file>
```

## Parse Tree Example

For input `id + id * id` with grammar1, the parse tree shows:
```
Parse Tree:
Expr -> Expr + Term
  Expr -> Term
    Term -> Factor
      Factor -> id
        id
  +
  Term -> Term * Factor
    Term -> Factor
      Factor -> id
        id
    *
    Factor -> id
      id
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| `make: command not found` | Install MinGW or use `gcc -std=c++11 -Wall -Wextra -o slr_parser src/*.cpp` |
| `slr_parser.exe not found` | Run `make` first |
| Empty output files | Test with valid input first |
| Parse tree empty | Only generated for successfully parsed inputs |
